# Sistema de Estacionamento usando Pilha

Projeto em **Java** que simula um sistema de estacionamento utilizando a estrutura de dados **Pilha (Stack)** e o conceito de **Tipo Abstrato de Dados (TAD)**.

O sistema permite:

* Guardar carros
* Remover o último carro inserido
* Remover um carro específico pela placa
* Listar os carros estacionados

A arquitetura é baseada em uma **interface (`Estacionamento`)**, garantindo abstração e facilitando futuras implementações com outras estruturas de dados.

Para remover um carro específico, é utilizada uma **pilha auxiliar**, respeitando as limitações da estrutura de pilha.

## Conceitos aplicados

* Programação Orientada a Objetos
* Interface e implementação
* Pilha (Stack)
* Pilha auxiliar
* Encapsulamento
* Entrada de dados com `Scanner`

Projeto desenvolvido com foco acadêmico para prática de **Estrutura de Dados** e **POO**.
